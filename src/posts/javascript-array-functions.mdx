JavaScript is becoming more and more powerfull when it comes to web and mobile app development. In fact they are 10 fundamental function in the array prototype that are really usefull. In oder to get to know about the existence of these function, we can open any modern web browser (chrome in this blog post). Open the console by inspecting any page. clear the console if needed then create a simple array:

```js
const myArr = [1,2,3,4]
```
```js
const myArr = [1, 2, 3, 4]
console.log(myArr)
// Outputs: [1, 2, 3, 4]

```
